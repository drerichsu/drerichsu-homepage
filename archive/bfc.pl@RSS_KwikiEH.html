RSS KwikiEH

The RSS feed is very flexible. You can actually autodetect the feed; truly slick RSS newsreaders, like NetNewsWire, allow you to drag the kwiki URL straight into the feed window and it will autodiscover the RSS link for you. Neat.

  • The feed URL is index.cgi?action=plugin&plugin=FeedEH&number=10&type=RSS
    • The number is optional (default is 10), and can be between 1 and 20.
    • The type is optional (default is RSS).
  • The Feed can be customized using a search= option. Almost all the search options described in Search Help are available. Yes, boolean, wildcards, keywords, etc.
    • Exception: you aren’t allowed to use ’&’ in search, though. Sorry; use AND.
    • for instance, to only subscribe to changes in the Home Page, add &search=(.pageid=homepage)
    • to subscribe to changes for all papers whose author is Tall, one might add &search=(.author=Tall and .type=papers)
  • You can also invoke it with the administration .cgi, and it will also reveal changes to pages marked private. (The regular feed will not.)
