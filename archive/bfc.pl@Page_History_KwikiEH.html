Page History KwikiEH

Every page has a Last Change to This Page link. If you want to go further into the history, you can turn on “Show advanced page edit history menu” in the Preferences.

The Show Page History command has some options too. One can compare any two versions of a page by editing the diff_revision= and current_revision= CGI options. if you leave off all the ‘revision’ settings, the default is to show the last change.
