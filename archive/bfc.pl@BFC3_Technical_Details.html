BFC3 Technical Details

DeletedPage BFC3 is built on Oddmuse. We customized the CSS (bfc.css) and installed several standard Oddmuse modules.

  • admin.pl
  • creole.pl
  • download.pl
  • drafts.pl
  • htmlcomment.pl
  • linebreaks.pl
  • localtime.pl
  • mac.pl
  • markup.pl
  • searchtags.pl
  • toc.pl

Then we added some custom modules:

  • latex-bfc.pl.
  • commentcount-bfc.pl. A customization of commentcount.pl with different words and comment formatting.
